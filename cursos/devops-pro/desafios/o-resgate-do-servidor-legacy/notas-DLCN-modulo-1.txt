### Parte 3: Produtividade e Segurança

cp /etc/ssh/sshd-config /tmp #- copia o arquivo para a pasta /tmp 
alias cadê='pwd'  #- cria um alias "atalho" para o comando pwd
mkdir PERIGO   #- cria o diretório Perigo

## 2. O que eu aprendi (explique com suas palavras)

- FHS: Estrutura de arquivos no Linux
- mkdir: Comando para criar diretório 
- ls: Comando para listar o conteudo de um diretório
- cd: Comando para mudar de diretório 
- rm: Comando para remover arquivo, serve também para remover diretórios (não vazios sem arumento) ou -r recursivo para remover diretorios
- alias: " atalho ou apelido " configurado geralmente no bashrc / arquivo de configuração do terminal do usuário 
- history: Comando que lê o arquivo bash_history do usuário onde esta a listagem dos últimos comandos executados
- Ctrl+R: Buscar comando no historico - crtl + r vai buscando o mais antigo.  
- /etc: diretorio de configuração do linux, onde geralmente ficam a maioria das configurações.  
- /var/log: Arquivos de log do sistema.

## 4. Respostas de Reflexão (Teoria)

1. Se o banco encher o disco, onde eu procuraria primeiro no FHS?
  Verificaria no /var/pasta do banco - geralmente é onde fica os dados do banco.

2. Diferença entre `cd ..` e `cd /`: 
  cd .. vai para o diretório anterior , cd / vai para a raiz do sistema 

3. Por que o Tab é uma medida de segurança?
  Não vejo como segurança, vejo apenas como um auxiliar para poupar tempo na procura do comando ou diretório correto e também de não errar comandos.
