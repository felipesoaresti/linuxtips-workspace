# O que é Cloud?

_Atualizado em 18/06/2026_

API - Padrão de comunicação Cliente servidor.  
Cloud - Serviços disponibilizados na internet
EC2 - VM Aws
S3 - Object Store - Pasta compartilhada via AWS 
IAM - Serviço responsável por segurança, autenticação ( Gerencia de tudo )
REGiÃO e ZONA - Região fisica (lugar do fisicamente/ país, local físico) / ZONA é equivalente a datacenter.
