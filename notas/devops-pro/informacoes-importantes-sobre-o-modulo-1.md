# Informações importantes sobre o módulo 1

_Exportado em 17/04/2026_

Servidor headless - sem periféricos ( remoto - ssh ) 
Hypervisors - Criar computadores virtuais dentro do computador físco, cada VM accha que tem o hardware independente. e o Hypervisor gerencia. 
- Bare-metal - Roda diretamente sobre o hardware, sem O.S. intermediário. Ex. KVM usado na AWS, VmWare ESX, Microsoft Hyper-Vserver, proxmox
- Hosted - É mais lento que o hypervisor, roda em cima do Sistema operacional, bom para laboratórios. Ex. VirtualBox, VmWare Workstation, Parallels

Cattle vs Pets (gado vs animal de estimação )
- Pets - Sysadmin tradicional, tudo na mão, bicho de estiumção, apego carinhoso 
- Cattle - Servidores não tem nome, são identicos, criados por automação, Moreu , nasce outro. Tudo gado
